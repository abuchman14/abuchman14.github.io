QT

Sprite Editor

A group project with Conner Schacherer, Safia Hassan, Faith Oladele, and Jeanette Arteaga The sprite editor features a grid on which users can draw. The grid size can be 8x8, 16x16, 32x32, or 64x64. Tools for drawing include: brush (default), erase, rectangle, line, fill, and stamp (heart, diamond, or triangle shape). The user can draw on multiple frames, which are displayed to create an animation of the user's drawings.

Other features: Saving/opening animations, Exporting animation as a .gif, Hotkeys for changing tools and adding frames

   

C#

Cryptography Methods Console Application

A console application which can be used to run 5 methods, listed below. All the methods are used in cryptography. Some could be used to help encrypt messages, and some (like quadratic sieve) could be used to help break encryptions and do some eavesdropping.

[Generators Mod P] Returns all numbers which, when raised to the powers 1 through p, generate all the numbers from 1 to p.

[Powers of g mod p] Prompts user for a number g and a modulus p. Calculates g to the all the powers 1 through p (mod p) and lists them.

[Number of elements in Z mod n] Calculates the number of elements in the set of integers mod n -- for any prime number p, this is p-1. More interesting for composite numbers, where the number of elements is fewer.

[Solovay Strassen Test] Runs the Solovay Strassen test on a given number n. (A probabalistic test to determine whether or not n is prime (or very likely prime))

[Quadratic Sieve Factoring] Factors (sqrt(n) + i)^2 for increasing values of i. Doesn't factor n for you, but helps with the process of finding factors using the quadratic sieve method. Available on gitHub here.
