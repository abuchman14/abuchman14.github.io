  1. Sprite Editor (QT)
    2. Details

    A group project with Conner Schacherer, Safia Hassan, Faith Oladele, and Jeanette Arteaga

    The sprite editor features a grid on which users can draw. The grid size can be 8x8, 16x16, 32x32, or 64x64. Tools for drawing include: brush (default), erase, rectangle, line, fill, and stamp (heart, diamond, or triangle shape). The user can draw on multiple frames, which are displayed to create an animation of the user's drawings.
    Other features:
    Saving/opening animations
    Exporting animation as a .gif
    Hotkeys for changing tools and adding frames (I should've used these more in the video above.)


    This was a school project which might be assigned to future students, so I can't make the code public online. I'd like to add an executable file at least, but this is strangely difficult in QT. I'll try again soon.
